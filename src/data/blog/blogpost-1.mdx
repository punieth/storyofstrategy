<h3 class="font-heading text-2xl font-bold">The Real Cost of Confusion</h3>
<p class="font-body mt-4">
  Meesho's mission is to democratize e-commerce. But for millions of first-time users, a confusing refund process isn't just a hassle—it's a breach of trust that costs a fortune in support overhead and lost customers. This is where user experience directly impacts the bottom line.
</p>

  <p class="text-center text-gray-600 mt-2 font-body">

  </p>

      <p class="text-4xl font-bold text-blue-600">~24M</p>

        "Where's My Refund?" Tickets Annually

    <div>
      <p class="text-4xl font-bold text-blue-600">~₹150</p>
      <p class="text-sm text-gray-600 font-body">Cost Per Support Ticket</p>
    </div>
    <div>
      <p class="text-4xl font-bold text-blue-600">~₹360 Cr.</p>
      <p class="text-sm text-gray-600 font-body">

    including the value of retaining churned customers.

<h3 class="font-heading text-2xl font-bold">The Game Plan: A Disciplined MVP</h3>
<div class="grid lg:grid-cols-2 gap-8 my-8">
  <div class="border-2 border-black rounded-lg p-6 shadow-[8px_8px_0_0_#000] bg-green-200">
    <h4 class="font-bold text-xl font-heading">
      What We're Building (The MVP)
    </h4>
    <ul class="mt-4 space-y-2 text-gray-700 list-disc list-inside font-body">
      <li>A read-only tracker screen.</li>
      <li>One new API endpoint to expose existing backend data.</li>
      <li>
        An event-logging pipeline to feed our new metrics dashboard.
      </li>
      <li class="font-semibold text-black">
        This is a quick win. Maybe 2 sprints, tops.
      </li>
    </ul>
  </div>
  <div class="border-2 border-black rounded-lg p-6 shadow-[8px_8px_0_0_#000] bg-red-200">
    <h4 class="font-bold text-xl font-heading">
      What We're *Not* Building (The Non-Goals)
    </h4>
    <ul class="mt-4 space-y-2 text-gray-700 list-disc list-inside">
      <li>NO real-time GPS courier tracking.</li>
      <li>NO in-app chat with support agents.</li>
      <li>NO ability to change refund destination.</li>
      <li class="font-semibold text-black">
        This discipline keeps us focused on the core problem.
      </li>
    </ul>
  </div>
</div>
