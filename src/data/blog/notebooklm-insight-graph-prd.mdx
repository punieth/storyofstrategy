    <div class="p-4 bg-blue-100 border-2 border-black mt-4">
        <h3 class="font-bold text-blue-800">Vision:</h3>

<Section icon="💡" number="7" title="Proposed Solution — MVP" color='bg-yellow-300'>
    <p><span class="font-bold">Concept:</span> A lightweight, trustworthy overlay that visualizes relationships via asynchronous, user-scoped semantic linking.</p>
    <div class="mt-4 p-4 bg-gray-50 border-2 border-black grid grid-cols-3 gap-4 text-center">
        <div><p class="text-2xl">⚡</p><p class="font-bold">Fast Loading</p><p class="text-xs">(&lt;200 ms)</p></div>
        <div><p class="text-2xl">🃏</p><p class="font-bold">Static Cards</p><p class="text-xs">w/ Snippets</p></div>
        <div><p class="text-2xl">🔍</p><p class="font-bold">Progressive Disclosure</p><p class="text-xs">3 → 10 Nodes</p></div>
    </div>
</Section>

<Section icon="🛡️" number="8" title="MVP Scope & De-Risking" color='bg-yellow-300'>
    <div class="space-y-4">
        <div class="flex items-start"><span class="text-xl mr-3">🐢</span><div><span class="font-bold">Latency/Cost:</span> Mitigated by asynchronous batch jobs.</div></div>
        <div class="flex items-start"><span class="text-xl mr-3">🔒</span><div><span class="font-bold">Privacy:</span> Mitigated by user-scoped, opt-in linking.</div></div>
        <div class="flex items-start"><span class="text-xl mr-3">🤖</span><div><span class="font-bold">Hallucination:</span> Mitigated by static cards with direct citations.</div></div>
        <div class="flex items-start"><span class="text-xl mr-3">🤯</span><div><span class="font-bold">Cognitive Overload:</span> Mitigated by progressive disclosure.</div></div>
    </div>
</Section>

<Section icon="🔒" number="11" title="Responsible AI & Privacy" color='bg-yellow-300'>
    <ul class="list-disc pl-5 space-y-2"><li>Links are computed only between user-approved notebooks.</li><li>All connections cite specific source paragraphs to be verifiable.</li><li>Confidence scores are displayed for each link.</li><li>Users can delete or dismiss irrelevant connections.</li></ul>
</Section>
